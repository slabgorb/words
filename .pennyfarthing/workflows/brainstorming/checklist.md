# Brainstorming Session - Definition of Done Checklist

## Session Setup
- [ ] Session topic clearly defined and understood
- [ ] Session goals articulated with specific outcomes
- [ ] Technique selection approach chosen (user-selected, AI-recommended, random, progressive)
- [ ] Selected techniques documented with rationale

## Idea Generation
- [ ] Minimum 50+ ideas generated (target: 100+)
- [ ] Multiple techniques executed with full facilitation
- [ ] Anti-bias protocol applied (domain pivoting every 10 ideas)
- [ ] Ideas captured using proper format (Category, Title, Concept, Novelty)
- [ ] Creative breakthroughs identified and highlighted

## Idea Organization
- [ ] All ideas organized into coherent themes
- [ ] Cross-cutting patterns identified
- [ ] Breakthrough concepts highlighted
- [ ] Implementation-ready ideas flagged

## Prioritization
- [ ] Top 3-5 high-impact ideas identified
- [ ] Quick wins separated from long-term investments
- [ ] Prioritization criteria applied (Impact, Feasibility, Innovation, Alignment)

## Action Planning
- [ ] Concrete next steps defined for priority ideas
- [ ] Resource requirements identified
- [ ] Success indicators established
- [ ] Clear pathway from ideas to implementation

## Documentation
- [ ] Session overview captured (topic, goals, approach)
- [ ] Complete idea inventory recorded by theme
- [ ] Prioritization results documented with rationale
- [ ] Action plans included with specific steps
- [ ] Session insights and creative breakthroughs preserved

## Quality Standards
- [ ] Ideas are genuinely novel (not just variations of obvious solutions)
- [ ] Themes reflect meaningful patterns (not forced categorization)
- [ ] Action plans are specific and achievable
- [ ] Documentation is comprehensive enough for future reference
