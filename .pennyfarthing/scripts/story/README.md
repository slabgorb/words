# Story Scripts

Scripts for story creation and management.

## Scripts

| Script | Purpose |
|--------|---------|
| `create-story.sh` | Create a new story in sprint YAML |
| `size-story.sh` | Estimate story points |
| `story-template.sh` | Generate story template |

## Usage

```bash
.pennyfarthing/scripts/story/create-story.sh
.pennyfarthing/scripts/story/size-story.sh
```

## Ownership

- **Primary users:** SM agent, `/pf-sprint story` commands
- **Maintained by:** Core Pennyfarthing team
